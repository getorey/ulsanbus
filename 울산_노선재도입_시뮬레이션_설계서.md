# 폐지 노선 재투입 효과 시뮬레이션 — 설계서
만차(차내 혼잡) 표시 + 123·126·307 재도입 시 혼잡 해소 효과를 before/after로 검토

## 1. 목표
- 현재 **만차(차내 재차인원이 많은) 구간**을 지도에 표시한다.
- 폐지 노선(123·126·307)을 **가정 배차간격으로 재투입**했을 때, 겹치는 기존 노선의 혼잡이 **얼마나 줄어드는지**를 before/after로 보여준다.
- **배차간격을 슬라이더로 조절**하며 투입 효율(대수 대비 해소량)을 확인한다.

## 2. 데이터로 가능/불가
- ✅ 만차 표시: 국토부 **노선별 차내 재차인원(cgst)** = 운행 1회차 차내 인원. 노선×정류장×시간대별 실측.
- ✅ 재투입 노선 경로: 123·126·307 정류장 시퀀스(복원 완료, `abolished_routes.json`).
- ✅ 흡수 가능 수요: 카드 **OD**(승차→하차) — 새 노선이 한 번에 연결하는 통행.
- ⚠ 가정 필요: **정원(좌석+입석)**, **전환율(새 노선으로 갈아탈 비율)**, **가정 배차**.
- ❌ 정밀 통행배정·유발수요·환승선택모형: 단일일자·합성데이터 한계로 미반영 → "효과의 방향성·규모 추정" 도구로 한정.

## 3. 만차(혼잡) 정의
- 혼잡률 = 차내 재차인원 ÷ **정원 C**(파라미터, 기본 45명: 시내버스 좌석+여유입석 가정).
- 등급: 여유(<50%) · 보통(50~80%) · 혼잡(80~100%) · **만차(≥100%, 재차≥C)**.
- 만차 구간 = 특정 노선·정류장·시간대에서 재차 ≥ C 인 구간. (지도에 빨강 강조)

## 4. 시뮬레이션 로직(단계)
1. **현황 혼잡 집계**: 노선 r, 정류장구간 s, 시간대 t 의 재차 `L[r,s,t]` (혼잡도 데이터).
2. **재투입 노선 R 정의**: 경로(정류장열) + 가정 배차 b분 → 시간대별 운행편수 `n_t = 60/b`.
3. **R 공급량(시간대별)**: `Cap_t = n_t × 정원C`.
4. **흡수 가능 수요**: 카드 OD 중 **출발·도착이 모두 R 경로상(±도보권)** 인 통행 = R가 한 번에 태울 수 있는 승객 `Dr_t`. (그 통행이 현재 어떤 기존 노선의 혼잡 구간을 지나는지 매핑)
5. **전환 수요**: `Shift_t = min(Dr_t × 전환율α, Cap_t)`.
6. **혼잡 해소 반영**: R와 **노선·구간이 겹치는 기존 노선**의 해당 구간 재차에서 `Shift_t` 를 차감 → `L'[r,s,t]`.
7. **효과 지표 산출(before/after)**:
   - 만차 구간 수: before → after (감소량/감소율)
   - 총 혼잡 초과인원(재차−C의 합) 감소량
   - 투입 효율 = 만차 해소 인원 ÷ **투입 대수**(=회전시간 ÷ 배차)

## 5. 조절 파라미터(슬라이더)
- **배차간격 b**(예 10~60분) → 투입 대수·공급량 변화
- **정원 C**(예 35~70)
- **전환율 α**(예 30~80%)
- 재투입 노선 선택(123/126/307, 복수 동시 가능)

## 6. 화면 구성(별도 파일 권장: `ulsan_route_sim.html`)
- 지도: 만차 구간(빨강) 표시 → 슬라이더 조절 시 해소되는 구간이 **빨강→주황/초록으로 완화**되는 애니메이션.
- 재투입 노선은 점선으로 오버레이(겹치는 기존 노선 강조).
- 우측 패널(before/after 비교):
  - 만차 구간 수: 32 → 21 (−34%) 같은 식
  - 총 혼잡초과 인원 감소
  - 투입 대수 / 효율(해소인원÷대수)
  - 시간대별 만차 구간 수 막대(전/후 겹쳐 표시)

## 7. 한계(반드시 명시)
- 정원·전환율·배차는 **가정값**(슬라이더로 민감도 확인).
- 통행배정·환승선택·유발수요 미반영 → **정밀 예측 아님**.
- 단일일자(2026-04-07) 스냅샷.
- cgst는 운행 1회차 재차의 표본 → 절대 정원 기준 만차 판정은 보수적으로.

## 8. 구현 단계
- **1단계**: 만차 구간 표시(혼잡도 기반) + 정원 슬라이더로 만차 임계 조절. (즉시 가능)
- **2단계**: 재투입 노선 흡수수요 매핑 + 배차/전환율 슬라이더 → before/after 만차 해소.
- **3단계**: 시간대별 전/후 비교 차트 + 투입 효율 지표.
