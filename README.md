## 🌐 공개 사이트 (GitHub Pages)

`docs/` 폴더를 GitHub Pages로 공개하면 아래 주소로 전체 결과를 볼 수 있습니다.

- **https://getorey.github.io/ulsanbus/**  ← `docs/index.html`(허브)에서 모든 화면으로 이동

> 공개 설정: GitHub 저장소 → **Settings → Pages → Source: `main` 브랜치 / `/docs` 폴더** 선택 → Save.
